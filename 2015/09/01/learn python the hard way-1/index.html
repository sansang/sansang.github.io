ex11 Asking Questions

raw_input()可以获取Terminal键入的数据，保存为字符串。int(raw_input())可将输入转为整型。input()也可以用于输入数据，但有安全方面的问题，并且在输入测试时，输入字符串报语法错误。

ex15 Reading Files
